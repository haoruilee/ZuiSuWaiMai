# 项目思路

MFC和数据结构结合，MFC模仿美团外卖UI，实现外卖核心功能

基于真实的学校地图，使用Dijkstra算法计算骑手的最佳路线并展示拥堵情况和预计时间

详细内容可参考开题和结题PPT
